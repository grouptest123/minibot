# 视频投喂说明

将需要分析的值班视频放到当前目录，推荐命名：

- `duty_watch_demo.mp4`

推荐录制脚本：

1. 固定摄像头，对准值班座位与人体上半身。
2. 前 3 秒保持空岗画面，作为空岗基线。
3. 第 4-12 秒人员进入座位并保持在岗。
4. 第 13-20 秒保持几乎不动，模拟静止过久。
5. 第 21-28 秒人员离开座位，模拟离岗。
6. 第 29-32 秒用纸张或手掌轻度遮挡镜头。
7. 第 33 秒后人员返回座位。

说明：

- 建议视频时长 30-45 秒。
- 保持摄像头固定，避免整体画面抖动。
- 值守区域尽量位于画面中心偏下区域。
- 放入新视频后，重启后端服务即可重新分析。
